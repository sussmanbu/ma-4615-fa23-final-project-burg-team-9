We tried a number of different exploratory data analysis, and here is one example. We believe GLM (“general” linear model) could explain our dataset better than other tools. This is the code we used:

load('dataset/traffic_violations.RData')
traffic_data_clean$Arrest_Type_Numeric <- ifelse(grepl("Marked", traffic_data_clean$`Arrest Type`), 1, 0)
model <- glm(Arrest_Type_Numeric ~ Race + Alcohol + Gender, data = traffic_data_clean, family = "binomial")
summary(model)

We first changed the categorical variable into a numeric variable of 0 and 1 to run the test. If the word “Marked” is contained, the value of 1 is assigned. If not, 0 is assigned. Based on Race, Alcohol, and Gender, we tried to predict the probability of Arrest Type. The intercept is 2.725248; this is the log-odds of the Arrest Type when all other predictor variables are zero.
